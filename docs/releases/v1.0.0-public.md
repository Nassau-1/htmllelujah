# HTMLlelujah 1.0.0

HTMLlelujah is an offline-first, AI-native presentation editor for Windows. It keeps
the editing model approachable for people used to slide software while storing decks
in an open, deterministic format that local tools and agents can inspect and change.

## What is included

- A native Windows editor with slide thumbnails, canvas editing, history, grouping,
  layers, locking, visibility controls, alignment guides, snapping, grids, themes,
  layouts, and reusable masters.
- Rich text, bullet lists, tables, images, shapes, connectors, icons, and round flag
  assets.
- Local `.hdeck` documents with atomic save, recovery, forward-version refusal, and
  portable embedded assets.
- Full-screen presentation mode plus standalone offline HTML and exact-page PDF
  export.
- Authenticated local-network collaboration with one authoritative host, ordered
  operations, reconnect handling, and visible expiring text locks.
- A packaged local MCP bridge for bounded reads, proposed mutations, explicit
  approval, commit, and undo workflows.

## Platform and installation

This release targets Windows 11 x64. The installer is deliberately named and
published as unsigned, so Windows may display a reputation warning. Verify the
SHA-256 checksum from the release assets before installing.

HTMLlelujah does not require an internet connection for ordinary editing,
presentation, save/reopen, or export workflows. Collaboration is local-network only;
there is no hosted relay or cloud account.

## Known boundaries

- Consumer synchronized folders transport deck snapshots but cannot provide a
  cross-machine writer lock. Choose one host and join its LAN session when editing
  together.
- Hosted templates, cloud comments, offline merge, animations, transitions, speaker
  notes, formulas, linked charts, advanced freeform paths, and boolean vector
  operations are outside this release.
- Exported HTML is a presentation output, not an editable interchange format.
- The supported compact editor layout starts at 1080 CSS pixels of width.
- Future unknown `.hdeck` schema versions are refused and left unchanged.

## Verifiable release assets

The release includes the Windows installer and blockmap, checksums, CycloneDX SBOM,
content inventory, candidate manifest, release evidence, and a deterministic final
release record. The final record binds the asset hashes, clean source commit, local
annotated tag object, remote annotated tag object, repository identity, and public
release notes. The publication workflow re-downloads every allowlisted asset and
checks its size and SHA-256 digest before it accepts the draft or public release.
